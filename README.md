# Customer Call Notes Generator

A simple web application that helps customer service representatives create professional call notes from customer interaction details.

## Purpose

This project was created to support customer service documentation, call center workflows, and professional communication. It helps users organize call details into a clear and consistent format.

## Features

- Enter customer name
- Enter reason for call
- Document actions taken
- Document resolution or next steps
- Generate professional call notes
- Beginner-friendly HTML, CSS, and JavaScript project

## Files

- `index.html` - Main web page
- `style.css` - Styling and layout
- `script.js` - Generates the call notes

## Example Use Case

A customer service representative can enter basic details from a customer interaction and quickly generate a clean summary for documentation.
